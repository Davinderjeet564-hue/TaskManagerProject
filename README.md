A simple task manager with add, edit, delete and complete task features. It uses React and TypeScript to create a responsive and user-friendly interface.

### Features
- Add a task 
- Edit a task 
- Delete a task 
- Complete a task 

### Tech Stack
- React
- Vite
- TypeScript
- Tailwind CSS

### Installation
```bash
git clone <repository-url>
npm install
npm run dev
```

### Usage

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```
